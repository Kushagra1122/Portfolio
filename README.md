# Portfolio

CLI-themed portfolio site (React + Vite + Tailwind). Dark/light theme and interactive terminal.

## Setup

- `npm install` then `npm run dev`
- **Resume**: Place your resume PDF at `public/MyResume.pdf` so the About page "Download Resume" link works. The file is served at `/MyResume.pdf`.

## Build

`npm run build` then `npm run preview` to test the production build.
